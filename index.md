# Days at Home *R* pipeline

This is an *R* reference pipeline for the Days-at-Home metric as oulined
here: <https://github.com/monash-neurointervention/uia-dah30>

This is still a work in progress.

# Get the package

Use `pak` to get the package from GitHub:

    pak::pak("agdamsbo/dahcalc")
    library(dahcalc)

When the package has matured, we are heading for CRAN.
