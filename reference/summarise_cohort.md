# Summarise DAH at the cohort level.

Summarise DAH at the cohort level.

## Usage

``` r
summarise_cohort(data, window_days = 30L)
```

## Arguments

- data:

  per‑patient result from \`compute_dah\`.

- window_days:

  observation window length (default = 30).

## Value

One‑row data.frame with summary statistics.
