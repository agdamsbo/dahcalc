# Package index

## All functions

- [`apply_window_and_death()`](https://agdamsbo.github.io/dahcalc/reference/apply_window_and_death.md)
  : Add observation window and flag deaths inside the window
- [`compute_dah()`](https://agdamsbo.github.io/dahcalc/reference/compute_dah.md)
  : Compute Days‑At‑Home per patient
- [`count_institutional_days()`](https://agdamsbo.github.io/dahcalc/reference/count_institutional_days.md)
  : Count institutional days that fall inside the (full) observation
  window
- [`extract_institutional()`](https://agdamsbo.github.io/dahcalc/reference/extract_institutional.md)
  : Extract hospital / rehabilitation stays (as integer days)
- [`extract_primary()`](https://agdamsbo.github.io/dahcalc/reference/extract_primary.md)
  : Extract the primary admission record
- [`merge_overlaps()`](https://agdamsbo.github.io/dahcalc/reference/merge_overlaps.md)
  : Merge overlapping or adjacent institutional intervals per patient
- [`plot_dah_distribution()`](https://agdamsbo.github.io/dahcalc/reference/plot_dah_distribution.md)
  : Plot the distribution of Days‑At‑Home
- [`run_dah_pipeline()`](https://agdamsbo.github.io/dahcalc/reference/run_dah_pipeline.md)
  : Run the whole DAH pipeline (dates returned as \`Date\` objects)
- [`summarise_cohort()`](https://agdamsbo.github.io/dahcalc/reference/summarise_cohort.md)
  : Summarise DAH at the cohort level (adds number of deaths)
- [`validate_events()`](https://agdamsbo.github.io/dahcalc/reference/validate_events.md)
  : Validate the long‑format event data
